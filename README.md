# Homebrew Tap for Ailoop

Install [ailoop](https://github.com/goailoop/ailoop) via Homebrew.

## Installation

```bash
brew install goailoop/cli/ailoop
```

## Usage

```bash
ailoop --version
```

For detailed usage and documentation, please refer to:
- GitHub: https://github.com/goailoop/ailoop

## Upgrading

```bash
brew upgrade ailoop
```

## Uninstalling

```bash
brew uninstall ailoop
brew untap goailoop/cli
```
